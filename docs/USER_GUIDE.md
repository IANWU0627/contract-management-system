## 3. 合同管理

### 3.1 合同列表

#### 3.1.1 筛选条件

| 条件 | 说明 |
|------|------|
| 关键词搜索 | 搜索合同名称、编号、相对方 |
| 合同类型 | 采购/销售/租赁/服务/劳动合同 |
| 合同状态 | 草稿/待审批/已批准/已签署/已归档/已终止 |
| 文件夹 | 按文件夹筛选 |
| 日期范围 | 按创建日期筛选 |

#### 3.1.2 操作按钮

| 按钮 | 功能 |
|------|------|
| 新建合同 | 创建新合同 |
| 批量审批 | 批量审批合同 |
| 批量删除 | 批量删除合同 |
| 导出 | 导出Excel |

### 3.2 创建合同

#### 步骤 1：选择合同类型

```
合同类型：
├── 📦 采购合同 - 企业采购原材料、设备等
├── 💵 销售合同 - 企业销售产品、服务等
├── 🏠 租赁合同 - 房屋、设备等租赁
├── 👔 劳动合同 - 员工劳动合同
├── 🛠️ 服务合同 - 提供或接受服务
└── 📄 其他合同 - 其他类型合同
```

#### 步骤 2：填写基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 合同名称 | 合同标题 | ✅ |
| 合同类型 | 选择合同类型 | ✅ |
| 合同金额 | 合同总金额 | ❌ |
| 开始日期 | 合同开始日期 | ❌ |
| 结束日期 | 合同结束日期 | ❌ |
| 相对方 | 签约方信息 | ✅ |

#### 步骤 3：填写相对方信息

```
相对方类型：
├── 甲方 - 合同发起方
└── 乙方 - 合同接收方

相对方字段：
├── 公司名称
├── 联系人
├── 联系电话
├── 电子邮箱
├── 统一社会信用代码
├── 地址
├── 开户银行
└── 银行账号
```

#### 步骤 4：选择模板（可选）

1. 点击"选择模板"
2. 浏览模板列表或搜索
3. 点击模板预览
4. 填写模板变量
5. 点击"使用模板"

#### 步骤 5：编辑合同内容

支持富文本编辑器，可：
- 设置标题格式
- 插入图片
- 添加表格
- 格式化文本

#### 步骤 6：上传附件

```
支持的文件类型：
├── PDF (.pdf)
├── Word (.doc, .docx)
├── Excel (.xls, .xlsx)
├── 图片 (.jpg, .png, .gif)
└── 压缩包 (.zip, .rar)

限制：
├── 单个文件大小 ≤ 10MB
└── 总文件数量 ≤ 20个
```

#### 步骤 7：设置标签

可选择已有标签或创建新标签。

## 5. 快速代码管理

### 5.1 快速代码概述

快速代码是一个灵活的枚举值管理系统，用于管理系统中的各种下拉选项、分类等。它支持：
- 多语言（中英文）
- 动态添加和修改
- 自定义排序
- 有效期管理
- 启用/禁用控制

### 5.2 快速代码应用场景

| 场景 | 说明 |
|------|------|
| 模板变量分类 | 管理模板变量的分类选项 |
| 合同类型字段 | 管理下拉选择框的选项 |
| 其他枚举值 | 任何需要动态配置的下拉选项 |

### 5.3 快速代码头管理

快速代码头是代码分类的容器。

#### 创建快速代码头

| 字段 | 说明 | 必填 |
|------|------|------|
| 名称（中文） | 分类的中文显示名称 | ✅ |
| 名称（英文） | 分类的英文显示名称 | ❌ |
| 代码 | 分类的唯一编码，如 `TEMPLATE_VARIABLE_CATEGORY` | ✅ |
| 描述（中文） | 分类的中文描述 | ❌ |
| 描述（英文） | 分类的英文描述 | ❌ |

### 5.4 快速代码项管理

快速代码项是具体的枚举值。

#### 创建快速代码项

| 字段 | 说明 | 必填 |
|------|------|------|
| 代码 | 代码项的唯一编码，如 `system` | ✅ |
| 含义（中文） | 中文显示名称，如"系统变量" | ✅ |
| 含义（英文） | 英文显示名称，如"System Variables" | ❌ |
| 描述（中文） | 中文描述 | ❌ |
| 描述（英文） | 英文描述 | ❌ |
| 标签 | 代码项分类标签 | ❌ |
| 生效日期 | 代码项生效开始日期 | ❌ |
| 失效日期 | 代码项失效日期 | ❌ |
| 是否启用 | 控制代码项是否可用 | ❌ |
| 排序 | 代码项显示顺序 | ❌ |

## 6. 变量管理

### 6.1 变量分类（使用快速代码）

变量分类现在使用快速代码管理，支持动态添加新分类，且支持多语言。

### 6.2 变量类型

| 类型 | 说明 |
|------|------|
| text | 文本输入 |
| number | 数字输入 |
| date | 日期选择 |
| textarea | 多行文本 |
| select | 下拉选择（可关联快速代码） |
| multiselect | 多选（可关联快速代码） |

### 6.3 变量列表

```
变量字段：
├── 变量编码 - 程序中使用
├── 变量名称（中文）- 中文显示名称
├── 变量名称（英文）- 英文显示名称
├── 变量类型 - text/number/date/textarea/select/multiselect
├── 所属分类 - 变量分组（使用快速代码）
├── 快速代码编码 - select/multiselect类型关联的快速代码
├── 默认值 - 可选的默认值
├── 必填标识 - 是否必填
├── 排序 - 显示顺序
├── 状态 - 启用/禁用
└── 描述说明 - 详细说明
```

### 6.4 创建变量

| 字段 | 说明 | 必填 |

| 变量编码 | 唯一标识，如 productName | ✅ |

| 变量名称（英文）| 英文名称，如"Product Name" | ❌ |
| 显示名称 | 前端显示，如"产品名称" | ❌ |
| 变量类型 | text/number/date/textarea/select/multiselect | ✅ |
| 所属分类 | 选择分类（使用快速代码） | ✅ |
| 快速代码编码 | select/multiselect类型关联的快速代码编码 | ❌ |
| 默认值 | 可选默认值 | ❌ |
| 描述说明 | 变量用途说明 | ❌ |
| 必填标识 | 是否必填 | ❌ |

## 🔧 常见问题

### Q1: 如何重置密码？

**管理员操作：**
1. 进入"用户管理"
2. 找到目标用户
3. 点击"重置密码"按钮
4. 确认重置（默认密码：123456）

### Q2: 如何添加新用户？

1. 以管理员身份登录
2. 进入"用户管理"
3. 点击"新增用户"
4. 填写用户信息：
   - 用户名
   - 昵称
   - 邮箱
   - 角色选择
5. 点击"保存"

### Q3: 如何自定义审批流程？

> 当前版本采用简化的审批流程

1. 提交合同时选择审批人
2. 审批人收到通知
3. 审批人审批通过/拒绝

### Q4: 如何导出合同数据？

1. 进入"统计报表"
2. 选择筛选条件
3. 点击"导出"
4. 选择导出格式（Excel）
5. 选择导出字段
6. 确认导出

### Q5: 模板变量如何工作？

1. 在"变量管理"中定义变量
2. 在"模板管理"中创建模板，插入变量 `{{变量名}}`
3. 创建合同时选择模板
4. 系统自动填充变量值
