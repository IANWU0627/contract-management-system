## Discoveries

1. **RoleManagement permissions error**: The `getActivePermissions()` API returns `{data: {list: [...], grouped: {...}}}` but `fetchPermissions` was setting `permissions.value = res.data || []` which made `permissions` an object instead of array.

2. **RoleManagement delete 500 error**: The `countByRoleId` SQL query was using `WHERE role_id = #{roleId}` but the `user` table has a `role` VARCHAR column (containing role codes like 'ADMIN'), not a `role_id` foreign key.

3. **QuickCode edit optimization**: When editing, the items need to be loaded from `GET /api/quick-codes/{id}` which returns `items` array.

4. **QuickCode update not saving items**: The update method only updated header info, didn't handle items CRUD.

3. **Create/Update**: Saves both language versions to database
